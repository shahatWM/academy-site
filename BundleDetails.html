  • Course 2

    Reusable Components

    22 Hrs

    • Lesson 1

      Overview

      WaveMaker enables developers to create and reuse custom components across apps, ensuring consistency and accelerating development.

    • Lesson 2

      Build a Prefab

      Prefabs in WaveMaker are reusable, customizable UI components bundled with functionality, allowing rapid integration across multiple apps for consistent and efficient development.

    • Lesson 3

      Prefab Life Cycle

      The life cycle of a WaveMaker Prefab involves creation, configuration, binding to app data, reuse across projects, and updates via version control—ensuring modularity and maintainability in app development.

    • Lesson 4

      Prefab Management and it's Usage

      Prefab management in WaveMaker involves creating, versioning, and publishing reusable UI-functional components that can be easily imported into applications—streamlining development, ensuring consistency, and promoting low-code modularity.

  • Course 3

    Builds and Debugging

    22 Hrs

    • Lesson 1

      Dev Tools

      WaveMaker development is supported by powerful dev tools like the Visual Page Designer, Service Designer, Data Modeler, API Designer, and integrated Debugger—enabling full-stack low-code development with drag-and-drop precision and real-time testing.

    • Lesson 2

      Logging and Profile Config

      In WaveMaker, logs are accessed via the Logs tab during preview or after deployment. Profiles are managed in Project Settings to define and switch between environment-specific configurations.

    • Lesson 3

      Build and Deploy Locally

      Builds and debugging in WaveMaker are streamlined through an integrated development environment that supports real-time previews, browser-based debugging, automated builds, and seamless deployment with logs and error tracing for efficient troubleshooting.

  • Course 4

    Customize, Extend, & Maintain

    22 Hrs

    • Lesson 1

      Understand Levels of Customisation

      WaveMaker supports multiple levels of code customization, including UI behavior with JavaScript, custom business logic using Java services, API integrations, and full-stack overrides through exported WAR files or custom components.

    • Lesson 2

      Maintain code without Studio

      WaveMaker allows you to maintain code outside the Studio by exporting the project as a ZIP or WAR, managing it in a Git repository, and using standard IDEs like Eclipse or VS Code for further customization, build, and deployment.
